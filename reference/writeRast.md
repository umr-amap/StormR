# Exporting rasters to GeoTIFF or NetCDF files

The `writeRast()` function exports rasters stored in SpatRaster objects
to GeoTIFF or NetCDF files.

## Usage

``` r
writeRast(rast, filename = NULL, path = "./", ...)
```

## Arguments

- rast:

  `SpatRaster` object.

- filename:

  character. Output file name. Can be either a `".tiff"` file (GeoTIFF)
  or `".nc"` (NetCDF). By default `filename=NULL` and the name of the
  file is generated based on raster information (storms and products)
  with the '.tiff' extension.

- path:

  character. Path to the directory where the file is exported to. By
  default `path=./'`.

- ...:

  Additional arguments to be passed to
  [[`terra::writeRaster`](https://rspatial.github.io/terra/reference/writeRaster.html)](https://rdrr.io/cran/terra/man/writeRaster.html)
  function when saving to `".tiff"` format.

## Value

`NULL`
